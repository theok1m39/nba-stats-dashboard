# 🏀 NBA Analytics Dashboard

A web-based NBA statistics and fantasy basketball tool built with HTML, CSS, and JavaScript.

## Features

### General Stats
- Search and compare any two NBA players side by side
- Visual bar chart comparison across key stat categories
- Highlights which player leads in each category

### Fantasy Category Rankings
- Select your league's scoring categories
- Instantly ranks all players by value for your specific league
- Supports standard Yahoo fantasy categories including PTS, AST, REB, 3PM, STL, BLK, FG%, and TO

### Trade Analyzer
- Add players to each side of a trade
- Analyzes trade value based on your league's categories
- Shows a category-by-category breakdown of what you gain and lose

## How to Use

1. Open the site and select a tab — General Stats, Category Rankings, or Trade Analyzer
2. For fantasy tools, select the categories your league uses
3. Search and add players, then hit Analyze
